# My Python Project

## Overview
This project is a Python application designed to analyze WhatsApp chat data. It provides functionalities to process and visualize chat information.

## Project Structure
```
my-python-project
├── src
│   └── main.py
├── requirements.txt
└── README.md
```

## Installation
To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd my-python-project
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```
python src/main.py
```
